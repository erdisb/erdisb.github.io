About

A bit about me

I'm a senior at Cornell University studying Operations Research & Engineering and Computer Science. I'm interested in the intersection of computing and finance — I'm drawn to problems where rigorous modeling meets real-world decision-making.

Professionally, I've built forecasting models at a credit fund, re-estimated derivatives risk for a major bank, and researched stochastic asset allocation under a financial engineering professor. These experiences taught me to think in distributions, build reliable data pipelines, and communicate technical ideas to stakeholders who care about the bottom line.

Away from the screen, I recharge by backpacking through remote trails, playing poker (both the math and the psychology), and training Muay Thai. I'm an aviation geek, a recreational basketball player and skier, and occasional philosophy reader. I believe the best engineers bring breadth — not just depth — to their work.

Projects

Things I've built

Live Equity Market Dashboard

A real-time streaming dashboard that ingests tick-level equity data from the Polygon.io WebSocket API, processes it through a lightweight event pipeline, and renders interactive candlestick charts with volume overlays and moving average bands.

Python WebSockets Redis Plotly Dash PostgreSQL

Poker EV Optimizer

A game-theory optimal (GTO) solver for simplified No-Limit Texas Hold'em scenarios. Uses counterfactual regret minimization to compute Nash equilibrium strategies across preflop and postflop decision trees, with an interactive visualizer for range charts and EV breakdowns.

Python NumPy Game Theory React D3.js

Flight Route Network Optimizer

An integer programming model that optimizes regional airline route networks given fleet constraints, passenger demand forecasts, and airport slot availability. Benchmarked against real schedule data from the Bureau of Transportation Statistics to validate cost savings of up to 9%.

Python Gurobi Pandas Linear Programming NetworkX

NBA Player Performance Predictor

A machine learning pipeline that forecasts player stat lines for upcoming NBA games using historical box scores, opponent defensive ratings, and rest-day features. Trained gradient-boosted and LSTM models, then served predictions via a FastAPI endpoint for daily fantasy lineup research.

Python scikit-learn PyTorch FastAPI PostgreSQL

Monte Carlo Options Pricer

A simulation engine for pricing exotic options (Asian, barrier, lookback) using geometric Brownian motion with variance reduction techniques including antithetic variates and control variates. Includes a Streamlit UI for adjusting parameters and visualizing convergence behavior.

Python NumPy SciPy Streamlit Monte Carlo
